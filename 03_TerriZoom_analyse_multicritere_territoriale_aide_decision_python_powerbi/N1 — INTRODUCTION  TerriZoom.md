# TerriZoom — Introduction

## Contexte

Les débats publics locaux reposent souvent sur des perceptions :
- sentiment d’insécurité
- perception de la pollution
- vision de la pauvreté

Ces perceptions peuvent être biaisées ou partielles.

## Problématique

Les indicateurs objectifs confirment-ils ou contredisent-ils ces perceptions ?

## Objectif du projet

Construire une analyse data basée sur des sources publiques pour :
- comparer des indicateurs territoriaux
- analyser leur évolution
- identifier des corrélations

## Indicateurs étudiés

- Criminalité
- Niveau de vie / pauvreté
- Qualité de l’air

## Approche

- Collecte de données multi-sources (CSV, API)
- Nettoyage et transformation
- Analyse exploratoire
- Visualisation
- Modélisation simple

## Limites

- hétérogénéité des sources
- granularité différente (commune / département)
- données manquantes possibles
