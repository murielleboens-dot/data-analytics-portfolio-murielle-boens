# Fact News Territoires — Sources de données et méthodologie

## 1. Criminalité

Source : données publiques (CSV)

- Niveau : département
- Variables :
  - nombre de faits
  - types d’infractions

Limites :
- pas toujours au niveau communal
- dépend des déclarations

---

## 2. Niveau de vie

Source : INSEE (déclarations de revenus)

- Niveau : commune
- Variables :
  - revenu médian
  - distribution

Limites :
- données déclaratives
- temporalité différente

---

## 3. Qualité de l’air

Source : API ATMO

- Indicateur : indice ATMO
- Niveau : commune

Avantages :
- indicateur standardisé
- multi-polluants

Limites :
- dépend des stations de mesure
- disponibilité variable

---

## Méthodologie

1. Collecte des données
2. Harmonisation des formats
3. Création de tables analytiques
4. Analyse exploratoire
5. Visualisation
